weapon handling instructions for the second scene:

Upon starting the program, the player will be able to select between three weapons using the keys "1", "2", and "3":

Weapon #1 corresponds to the parabolic shot.
Weapon #2 corresponds to the magnetic shot.
Weapon #3 corresponds to the bouncing shot.
Each weapon has a fire rate that can be modified from the ScriptableObject, along with various other customizable components.
